# 家政家洁管理系统 java320

## 项目概述

家政家洁管理系统是基于SpringBoot和Vue的前后端分离架构，旨在提供高效、便捷的家政服务管理解决方案。

## 技术栈

- **后端**: SpringBoot
- **前端**: Vue
- **数据库**: MySQL 8, Redis

## 功能模块

- 用户管理
- 服务项目管理
- 预约管理
- 订单管理
- 账户与财务管理

## 系统角色

- 系统管理员
- 家政服务人员
- 客户

## 运行环境

- **后端**: 
  - JDK 1.8 或更高版本
  - Redis 3.2 或更高版本
- **前端**: 
  - 现代浏览器（推荐使用Chrome或Firefox）
- **数据库**: 
  - MySQL 8.0 或更高版本

## 部署步骤

1. 准备运行环境
2. 部署后端服务
3. 部署前端应用
4. 启动Redis服务
5. 初始化数据库

## 目录结构

```
家政家洁管理系统/
├── backend/              # 后端代码
├── frontend/             # 前端代码
└── database/             # 数据库脚本
```

## 核心亮点

- **可扩展性**: 系统模块化设计，便于后期功能扩展和系统维护。
- **可靠性与安全性**: 严谨的权限控制，确保系统稳定和数据安全。
- **前后端分离**: 独立部署的前后端，可灵活应对不同场景需求。
